Source Code

Source code is the most essential part of a program, of software. For, it is normal to consider that the machine performs what programmers know as programming language, but what is interpreted it's actually not the programmer's source code. The programmer's source code, is the character set, whether number, letter, or special, that he so that a series of actions are taken on the Computer. It's importance lies in the simplest things, there is no need to necessarily cite applications, programs, because what the source code does in people's lives.

The source code then works as follows:

  • Simple code: Simple codes such as HTML and JavaScript can be executed, that is, interpreted by the computer, without the need to be compiled

  • Código-complexo: quando os códigos são difíceis de serem interpretados, teremos que dividir em um processo de três etapas, o primeiro é o código fonte escrito pelo programador

IDE - Integrated Development Environment

    IDE is a program used to develop Software, that unites certain features assisting in software development, this helps in the productivity of the developer. It should be able to rotate or compile the codes, leaving a certain system executable and must identify not only syntax errors but also some errors that occur during the execution of the program. For the IDE to be complete it needs a code editor, a compiler that will make the source code executable, a link that links various parts of the source code and helps the compiler make your program one thing only, a debug that assists in the process of finding and correcting defects in the program, a modeler towards interface creation, a code generator that will generate automatic codes facilitating the life of the programm deploy which is the distribution of the Software to the installer or to the cloud, automated tests that Performs testing in the software avoiding more common problems and the spectrum consisting of the constant improvement of the source code.
